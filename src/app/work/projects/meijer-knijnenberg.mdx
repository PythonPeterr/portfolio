## Overview

Developed a custom website for Meijer & Knijnenberg, a recruitment agency, featuring a job board and an administrative upload portal that allows them to easily manage and publish job listings while enabling candidates to apply directly through the website.

## Key Features

- **Job Board**: Public-facing job listings page displaying all available positions with detailed descriptions and requirements.
- **Upload Portal**: Administrative interface allowing Meijer & Knijnenberg staff to upload and manage job listings without technical knowledge.
- **Application System**: Integrated application functionality enabling candidates to apply for positions directly through the website.
- **Responsive Design**: Mobile-friendly interface ensuring accessibility across all devices for both job seekers and administrators.

## Technologies Used

- **Frontend**: Modern web technologies for a clean, responsive user interface
- **Backend**: Server-side functionality for job listing management and application processing
- **Database**: Secure storage for job listings and candidate applications
- **File Upload**: Robust file handling for job descriptions and applicant resumes

## Challenges and Learnings

The main challenge was creating an intuitive upload portal that non-technical staff could use confidently while maintaining data consistency and security. The solution involved designing a simple, user-friendly interface with clear validation and helpful error messages.

## Outcome

The website successfully streamlined Meijer & Knijnenberg's job posting process, eliminating the need for technical intervention when publishing new positions. The integrated application system improved candidate experience and made the recruitment workflow more efficient.
